# Legend of Bahamut Strategy

バハムート大陸を舞台にした、ブラウザで遊べる本格ファンタジー戦略シミュレーションゲームです。
プレイヤーはマスター（指揮官）となり、拠点を制圧しながら大陸全土の統一を目指します。

## 🎮 ゲームの特徴

- **戦略マップとリアルタイム進軍**: マップ上の拠点をクリックして部隊を移動させます。敵部隊と接触するとヘックス（六角形）タイルのバトル画面へ移行します。
- **ユニット成長（ランクシステム）**: 戦闘を繰り返すことでユニットが経験値（練度）を獲得し、**RANK E** から最高 **RANK S** まで成長します。ランクアップによりHPと攻撃力が上昇します。
- **拠点統治と経済**: 拠点を制圧することでターンごとの収入が増加します。獲得した資金（G）を使って新しい部隊を編成したり、ユニットを雇用・強化したりできます。
- **個性あふれるマスターとユニット**: 「ジーク（聖騎士）」「アーシェ（女王）」「クリムゾン（魔王）」の3人のマスターから選択可能。各マスターには専属の兵種が存在します。
- **拠点の特殊兵**: 特定の拠点を制圧すると、そこでしか雇用できない「拠点固有兵（ケルベロス、魔導師など）」を利用できるようになります。

## 🕹 操作方法

### 戦略マップ
- **左クリック**: 自軍部隊を選択し、隣接する拠点へ移動指示を出します。
- **右クリック**: 各拠点の詳細メニューを開きます（拠点メニュー内では雇用や強化が行えます）。
- **ターン終了ボタン**: 自分の行動が終わったらターンを終了させます。

### バトル画面
- **左クリック**: ユニットを選択し、移動範囲（青）をクリックして移動、または攻撃範囲（赤）内の敵をクリックして攻撃します。
- **ターン終了ボタン**: 全ユニットの行動が終わるか、手動でターンを終了させます。
- **撤退ボタン**: 戦場から離脱し、王都へ帰還します（ペナルティあり）。

## 🛠 技術スタック
- **Frontend**: HTML5, Vanilla JavaScript, CSS (Tailwind CSS)
- **Graphics**: HTML5 Canvas (戦略マップ), DOM-based Hex Grid (バトル)
- **Font**: Google Fonts (DotGothic16)

## 🚀 開発者向け
本作はシングルファイルのHTMLとして完結しており、ブラウザで `game.html` を開くだけで即座にプレイ可能です。
内部には `TestSuite` が組み込まれており、開発時のロジック検証が容易になっています。
